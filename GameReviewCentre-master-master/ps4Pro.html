Review

It is not the greatest improvement from the PS4 but it still does have a few improvements over the PS4. These are that the PS4 Pro now supports 4k output, it has an extra 1GB specifically put in for better graphics so it is no longer shared by the existing 8GB GDDR5.

Also the PS4 Pro processor is almost double the processing power of the PS4. The graphics has over double the graphics of the PS4 which has 1.84 TFLOP AMD Radeon and the PS4 Pro has 4.2 TFLOP AMD Radeon.
